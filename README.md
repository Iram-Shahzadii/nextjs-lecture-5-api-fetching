## 📌 Project Overview

This project demonstrates how to build a **data-driven application** using Next.js App Router with TypeScript.  
It fetches user data from an external API and provides **real-time search filtering** along with **dynamic user profile pages**.

---

## 💡 Key Learning Outcomes

- 🔗 **API Integration**  
  Fetching external data using `fetch()` with async/await  

- 🔀 **Dynamic Routing**  
  Creating dynamic pages using `[id]` routes  

- ⚙️ **State Management**  
  Managing UI state with `useState` and `useEffect`  

- 🔍 **Live Search Filtering**  
  Real-time filtering based on user input  

- 🧠 **TypeScript Usage**  
  Strong typing with custom interfaces for API data  

---

## 🛠 Tech Stack

- **Framework:** Next.js (App Router)  
- **Library:** React.js  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **API:** JSONPlaceholder  
- **Deployment:** Vercel  

---

## ✨ Features

- 🔍 Live search filter (instant results)
- 📱 Fully responsive UI (mobile + desktop)
- 🔗 Dynamic user profile pages
- ⏳ Loading & error handling states
- 🎨 Clean modern UI with hover effects
